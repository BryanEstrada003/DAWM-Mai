### Instrucciones

* Desde el editor de texto, modifica el archivo `public/stylesheets/ejercicio.css`, 
	+ Agrega las etiquetas solicitadas dentro de cada uno de los comentarios marcados, con:

* Pruebas unitarias

	+ Ejecutar: `npm test`
